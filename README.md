# Hamaara Technologies

Dark-theme website — 16 pages, zero dependencies, no build step required.

## Deploy to GitHub Pages

1. Create a new **public** repo on GitHub (e.g. `hamaara-tech`)
2. Push this folder to the `main` branch
3. Go to repo **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Site will be live at `https://<your-username>.github.io/hamaara-tech/`

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```
